# Healthcare Analytics Capstone

## Overview
 Transformed raw healthcare data into meaningful insights related to patient visits, demographics, and regional trends.  These visualizations were critical for converting raw healthcare data into actionable insights aligned with Aura’s analytics goals.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Key Visualizations / Results
- Used barplots to compare category frequencies, a heatmap to reveal feature correlations, and box plots to visualize distribution differences across groups
- After reviewing all the visualizations, the heatmap indicates that healthcare visits are strongly associated with hospital stays, chronic conditions, and emergency occurrences. Additionally, most visits were concentrated among patients living in the West region, and a large portion of these visits were linked to individuals reporting poor health.

## How to Run
Open the Jupyter notebook and run all cells.
