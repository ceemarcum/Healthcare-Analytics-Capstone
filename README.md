# Healthcare Analytics Capstone

## Overview
This project analyzes healthcare utilization patterns using Python to uncover insights about patient visits, demographics, chronic conditions, and regional trends.

## Key Visualizations
- **Barplots:** Compared visit frequencies across regions and demographics.
- **Heatmap:** Identified correlations between chronic conditions, emergency visits, and health status.
- **Box Plots:** Visualized distribution differences and detected outliers in visit behavior.

## Why It Matters
These visualizations transform raw healthcare data into actionable insights that support data-driven decisions in patient care, resource allocation, and regional health planning.

## Tools
Python • Pandas • Seaborn • Matplotlib

## How to Run
Open the Jupyter notebook and run all cells.
